Diabetes Prediction Using Machine Learning

Developed a machine learning model to predict diabetes risk using patient health indicators including glucose level, BMI, age, blood pressure, and insulin measurements.

The project involved data cleaning, exploratory data analysis, feature engineering, model training, and evaluation using classification metrics.

Key Insights

  • Glucose level was the strongest predictor of diabetes.
  • BMI and Age showed significant influence on prediction outcomes.
  • The model achieved approximately 75% prediction accuracy.
  • Precision and recall metrics highlighted opportunities for further model improvement.

Tools: Python, Pandas, Scikit-Learn, Matplotlib
