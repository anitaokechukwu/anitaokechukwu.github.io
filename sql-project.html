Hospital Encounter Data Analysis

Project Overview

This project analyzes hospital encounter data using SQL to identify patient trends, healthcare utilization patterns, treatment costs, and readmission rates. The goal was to generate insights that can support operational efficiency and improve patient care.

Tools Used

  • SQL
  • MySQL Workbench
  • GitHub

Project Gallery

Key Questions Answered

  • Which encounter types were most common?
  • What was the average cost per encounter?
  • Which patients had the highest readmission rates?
  • What age groups utilized healthcare services most frequently?
  • Which conditions generated the highest healthcare costs?

Key Insights

  • Outpatient encounters accounted for the majority of visits.
  • Readmission rates were higher among patients with chronic conditions.
  • Certain age groups utilized healthcare services significantly more than others.
  • A small number of conditions contributed disproportionately to total healthcare costs.

Recommendations

  • Strengthen follow-up programs for high-risk patients.
  • Improve preventive care initiatives for chronic diseases.
  • Monitor high-cost conditions to optimize resource allocation.
  • Develop targeted interventions to reduce avoidable readmissions.
