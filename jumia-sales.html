📌 Project Overview

This project analyzes retail sales data using SQL to identify sales trends, customer behaviour, product performance, and revenue opportunities. The objective was to answer business questions using SQL queries and transform raw data into actionable insights.

🎯 Business Problem

Management wanted to understand sales performance, identify high-value customers, determine best-selling products, and improve business decisions using data.

📂 Dataset

  • Retail sales transactions
  • Customer information
  • Product information
  • Order details

🛠 Tools Used

  • SQL
  • MySQL Workbench
  • GitHub

❓ Business Questions

  • Total Revenue Generated
  • Top Selling Products
  • Best Customers
  • Monthly Sales Trend
  • Average Order Value
  • Customer Purchase Frequency

💻 SQL Highlights



SELECT
product_name,
SUM(sales) AS TotalSales
FROM orders
GROUP BY product_name
ORDER BY TotalSales DESC;

More SQL queries are available in the GitHub repository.

💡 Key Insights

  • Top products generated most of the revenue.
  • Sales peaked during seasonal promotions.
  • A small percentage of customers contributed a large share of sales.
  • Some products consistently underperformed.

✅ Recommendations

  • Increase inventory for high-performing products.
  • Improve marketing for underperforming products.
  • Develop customer loyalty programmes.
  • Monitor monthly sales trends for better forecasting.
